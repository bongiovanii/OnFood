# 🍳 OnFood

O **OnFood** é uma aplicação web que ajuda você a descobrir receitas a partir dos ingredientes que você tem em casa.  
Basta informar os ingredientes disponíveis na sua geladeira e o sistema sugere uma receita que pode ser preparada com eles.

A proposta do projeto é facilitar o dia a dia na cozinha, evitando desperdícios e ajudando usuários a encontrarem novas receitas de forma prática.

---

## 📦 Como Rodar o Projeto Localmente

Siga os passos abaixo para executar o projeto na sua máquina.

### 1. Instalar o XAMPP
Baixe e instale o **XAMPP** em sua máquina caso ainda não possua.

### 2. Iniciar os serviços
Abra o **XAMPP Control Panel** e inicie os seguintes serviços:

- **Apache**
- **MySQL**

### 3. Abrir o projeto na IDE
Abra o projeto **OnFood** na sua IDE de preferência (ex: IntelliJ, Eclipse ou VS Code).

### 4. Executar a aplicação
Dentro do projeto, localize a **classe `Application` do Java`** (classe principal da aplicação).

Execute essa classe para iniciar o servidor da aplicação.

### 5. Acessar no navegador
Após iniciar a aplicação, abra seu navegador e acesse:
localhost:8080


A página do **OnFood** será carregada e você já poderá utilizar o sistema.

---

## 💡 Como Usar

1. Acesse a aplicação pelo navegador.
2. Informe os ingredientes disponíveis.
3. O sistema irá sugerir uma receita baseada nesses ingredientes.

Este projeto foi desenvolvido com o objetivo de aplicar conceitos de desenvolvimento web e integração entre **backend em Java** e **banco de dados**, criando uma aplicação útil para o dia a dia.
